#### Search Algorithms:
- `A*`
- `Greedy`
- `Uniform-cost`

#### For `A*` and `Greedy` next heuristic functions are available:
- `Misplaced` `--heuristic=misplaced`
- `Manhatten` `--heuristic=manhatten`
- `Euclidian` `--heuristic=euclidian`
